From data visualization to coding environments, these tools can help you analyze your findings and develop your projects.

  • Spreadsheet Software & Data Management:
    • Google Sheets (Free, web-based)
    • Microsoft Excel (Tutorials & Templates)
    • Apache OpenOffice Calc (Free desktop alternative)
    • LibreOffice Calc (Another free desktop alternative)
  • Statistical Analysis & Programming:
    • R Project for Statistical Computing (Free, powerful)
    • RStudio Desktop (IDE for R)
    • Python (General purpose programming, vast science libraries)
    • Jupyter Notebooks (Interactive computing with Python/R)
    • Google Colaboratory (Free cloud Jupyter notebooks)
    • JASP (Free & open-source statistical software, user-friendly)
    • GNU PSPP (Free alternative to SPSS)
    • SciPy (Python library for scientific computing)
    • Pandas (Python library for data manipulation)
  • Data Visualization & Graphing:
    • Google Charts (Web-based charts)
    • Tableau Public (Free data visualization software)
    • Matplotlib (Python plotting library)
    • Seaborn (Python statistical data visualization)
    • Plotly (Interactive graphing libraries for Python, R, JS)
    • Flourish (Easy data visualization & storytelling)
  • Design, Simulation & Prototyping:
    • TinkerCAD (Free, easy 3D design for beginners)
    • Autodesk Fusion 360 (Free for students, advanced CAD)
    • Arduino IDE (For electronics prototyping)
    • Scratch (Visual programming for simulations/games)
    • Blender (Free & open-source 3D creation suite)
    • SimScale (Cloud-based simulation platform, academic access)
  • Version Control:
    • Git (Version control system)
    • GitHub (Platform for hosting code with Git)
    • GitHub Hello World Tutorial
